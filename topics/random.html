PRIME NUMBER DETECTOR

Input a number to determine if it is prime.

The maximum input value before truncation error is 9007199254740991 (~9e15, nine and fifteen zeros!). This is because JavaScript uses double-precision floating-point numbers according to IEEE 754.

UNCLE GIMPY'S INTELLECTIFIER (WALLET REFERENCE SHEET)

Here is an old reference sheet with simple formulas for length, area, volume as well as useful conversions. It's called Uncle Gimpy's Wallet Intellectifier, and is meant to be folded up and kept in a wallet (or purse). This is certainly from a time before cell phones (at least the 1980s), but these sorts of reference sheets are still just as useful today, if not less common.
It is mostly legible in it's current state, but I intend to retype and upload a clean copy in the future.
